# GitHub-Trace

A Web Application based on Shadcn UI and Next.js, used to find the connection relationship between usernames and email addresses through GitHub commit history.

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FM1r0ku%2FGithub-Trace)

## Development

```bash
$ npm install
$ npm run dev

```

Open in browser [http://localhost:3000](http://localhost:3000)
